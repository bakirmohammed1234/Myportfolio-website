Featured Projects

Some of my recent work

Synthetic Data Generation & LLM Fine-Tuning Pipeline

I built a pipeline to generate synthetic instruction–answer data from documents and automatically evaluate sample quality using an LLM-based scoring module. I formatted the data for supervised training and fine-tuned the Llama 3.2-1B model with LoRA and 4-bit quantization to create a domain-specialized LLM.

Python PyTorch Hugging Face Transformers LORA

AI-Powered News Research Tool

I built an end-to-end AI-powered news research tool using a Retrieval-Augmented Generation (RAG) pipeline with LangChain. I implemented document ingestion workflows with TextLoader and UnstructuredURLLoader, and created a FAISS vector database to enable fast semantic search and context retrieval. I then developed a RetrievalQA system capable of answering user queries with cited sources.

Python LangChain FAISS Streamlit

End-to-End Customer Churn Prediction

I developed an end-to-end customer churn prediction system, starting with data cleaning, feature engineering, and exploratory analysis. I trained and evaluated machine learning models to identify customers at risk of leaving, and visualized the results to provide actionable business insights. The final solution was deployed as a web application using Flask.

Scikit-learn Python Flask Pandas & NumPy & Matplotlib Python

COVID-19 Prediction System

Developed a machine learning model using Random Forest to predict patient death risk from COVID-19. Integrated the model with FastAPI for production deployment and created an Angular frontend for user interaction.

Python Scikit-Learn FastAPI ReactJs Cassandra

Building Energy Efficiency Prediction
